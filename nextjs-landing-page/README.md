# nextjs-landing-page

A comprehensive skill for building production‑ready Next.js landing pages, inspired by the [CodeBridge landing page](https://codebridge.dev).

## Features

- 🌍 Internationalisation (i18n) with Lingui
- 🧪 A/B testing with cookie‑based variant assignment
- 📊 UTM tracking (first‑touch attribution)
- 📝 Waitlist signup with Turso database and live counter
- 🎨 Beautiful UI with Tailwind CSS and shadcn/ui
- ✨ Smooth animations with Framer Motion
- 🧩 Modular component architecture
- 📱 Fully responsive
- 🔍 SEO‑friendly with dynamic sitemap/robots
